Тестовый проект Spring rest выводит данные про работников в формате JSON
Стэк проекта: 
1) Java 8
2) Maven
3) Hibernate
4) MySql
5) Аннотации @Service, @Repository, @ControllerAdvice, @ExceptionHandler, @RestController
6) HTTP запросы такие как GET, POST, PUT, DELETE
